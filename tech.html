Ecommerce Project V1 - "STAY QUIET"

View the full site.

I chose this work sample for my portfolio because I went all out. I say this because I went out of my comfort zone, tried new things, and implemented newfound knowledge. This project focused mostly on HTML and CSS but I implemented a small amount of Javascript. This project challenged me to do something different. It was my first time making an ecommerce site and I tried new things like an email subscription section, products page, sort by button, and I made a new contact page using grid which was exciting. If I had more time to work on this project I would have worked on the media queries to make it usable on phones. This project influenced me to continue to grow my coding skills because I want to be able to do Microsoft, Nike and Apple level websites by the end of the apprenticeship.
