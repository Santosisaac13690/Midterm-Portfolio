Ecommerce Project - "STAY QUIET"

I chose this work sample for my portfolio because I went all out, I say this because I went out of my comfort zone, tried new things, & implemented newfound knowledge. This project focused mostly on HTML & CSS but I implemented a small amount of Javascript. This project challenged me to do something different, It was my first time making an ecommerce site & I made things I never made before like an email subscription, products page, sort by button, & I made a new contact page form which was exciting. This project influenced me to continue to grow with my coding skills, I want to be able to do Microsoft, Nike, Apple level websites by the end of the apprenticeship.
